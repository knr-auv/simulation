# AUV Virtual Environment 2.0
Oko≈Ñ AUV Virtual Environment 2.0 is built in [Unity](https://unity.com/) 2020.1.3f1. Simulates behaviour of the Okon. It's an AUV built by [KNR](http://knr.meil.pw.edu.pl/). VE includes:
- underwater shaders
    - color correction
    - physical camera setup
    - motion blur
- simple fluid dynamics
    - laminar and dynamic flow
    - buoyancy
- live video stream
  - color camera
  - depth for point cloud
- level based mission simulation
- generating dataset for object detection neural net
